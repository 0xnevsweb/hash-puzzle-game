# Development

## Scripts

- `npm run dev` — Vite dev server (port 3000, hot reload).
- `npm run build` — production build emitted to `dist/`.
- `npm run preview` — preview the built bundle locally.

## Conventions

- Two-space indent, single quotes, semicolons (see `.prettierrc.json`).
- Keep modules focused: pure logic in `src/utils/`, presentational code in
  `src/`.
- Prefer plain CSS — no Tailwind/styled-components/etc.
